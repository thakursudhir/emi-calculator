# 💰 EMI & Interest Calculator

A simple and responsive **EMI & Interest Calculator** built using **HTML, CSS, and JavaScript**.
This tool helps users calculate **EMI, total interest, and total payment** using either the **Standard EMI (Reducing Balance)** method or **Simple Interest** method.

# ✨ Features

* 📊 Calculate **EMI**
* 💰 Calculate **Total Interest**
* 🧾 Calculate **Total Payment**
* 🔁 Two calculation methods:

  * **Standard EMI (Reducing Balance Method)**
  * **Simple Interest (Flat Rate Method)**
* ⚡ Instant calculation using JavaScript
* 📱 Responsive and clean UI
* 🎯 Beginner-friendly project

---

# 🧮 Inputs Required

Users need to enter the following values:

* **Loan Amount (₹)**
* **Interest Rate (% per year)**
* **Loan Tenure (Years)**
